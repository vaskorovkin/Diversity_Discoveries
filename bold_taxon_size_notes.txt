BOLD global public data size notes
==================================

These are approximate global BOLD summary counts from API queries run on
2026-04-29. Counts are for public BOLD records matching the taxon query, not
for all sequence data worldwide. Some groups are nested inside others.

| Group | Records | Coord coverage | Usefulness |
|---|---:|---:|---|
| Plantae | 468K | 41% | Important for medicinal/natural-products angle, but geography weaker |
| Fungi | 347K | 25% | Discovery-relevant, but geography weak |
| Arthropoda | 18.8M | 94% | Huge; must split |
| Mollusca | 302K | 45% | Useful, especially marine/freshwater |
| Gastropoda | 212K | 47% | Manageable mollusk subgroup |
| Bivalvia | 69K | 39% | Manageable |
| Coleoptera | 1.2M | 88% | Large but feasible if split |
| Formicidae | 383K | 93% | Very feasible first global insect group |
| Araneae | 222K | 83% | Very feasible spider group |

Important caveat:
The 1 million record cap in the BOLD Portal API is per query/token, not a
time-resetting allowance. Larger groups need to be split by country, taxonomic
subgroup, or another query dimension.
