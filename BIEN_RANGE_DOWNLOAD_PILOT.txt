Generic pattern for a new batch:

1) Availability-only
Rscript Scripts/download/18_bien_range_download_pilot.R \
  --batch-id <N> \
  --rank-start <START_RANK> \
  --top-n <BATCH_SIZE> \
  --outdir Data/raw/bien/batches/batch_<NNN>_ranks_<START>_<END> \
  --availability-only

2) Download
Rscript Scripts/download/18_bien_range_download_pilot.R \
  --batch-id <N> \
  --rank-start <START_RANK> \
  --top-n <BATCH_SIZE> \
  --outdir Data/raw/bien/batches/batch_<NNN>_ranks_<START>_<END> \
  --overwrite

3) Optional: build skinny ranges and a local richness raster for a batch
Rscript Scripts/download/18_bien_range_download_pilot.R \
  --batch-id 1 \
  --rank-start 1 \
  --top-n 5000 \
  --outdir Data/raw/bien/batches/batch_001_ranks_000001_005000 \
  --template-raster /absolute/path/to/template_raster.tif \
  --overwrite

Notes:

- The current completed sweep covers the full canonical species-like pool.
- It records timing (`availability_seconds`, `download_seconds`) and file sizes (`total_bytes`, `shp_bytes`, etc.) in `summary.csv`.
